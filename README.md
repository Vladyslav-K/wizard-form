# Wizard form

---

## Step-by-step registration form

### Created using react, redux, redux-saga and formik

### [Link on Github Page](https://vladyslav-k.github.io/wizard-form)

---

### Install dependencies:

```
$ npm install
```

### Build for production:

```
$ npm run build
```

### Build for development:

```
$ npm run start
```

### Run test

```
$ npm run test
```
